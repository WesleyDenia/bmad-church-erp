curso-bmad

Direções visuais para uma home operacional por perfil.

Este showcase explora seis caminhos visuais para o produto: todos respeitam a mesma fundação de UX, mas variam no peso visual, tom emocional e linguagem cromática. O objetivo é escolher a direção que melhor transmite confiança calma, clareza e uso acessível para voluntários.

Como avaliar

O que observar

  • Hierarquia dos blocos e leitura das prioridades
  • Densidade: confortável sem ficar vazia
  • Tom: sóbrio, acolhedor e não corporativo
  • Facilidade para usuários não técnicos
  • Potencial para tesouraria e secretaria
